# Velox App

`velox-app` is the future working software application for Velox MVP 1.

The foundation repository remains `velox-technical-foundation`.

This app structure is intentionally minimal. It is prepared for the first useful Velox loop:

```text
Add creator -> Enter metrics -> Score creator -> View profile -> Generate report
```

No external APIs, Supabase connection, authentication, scraping, or payment features are included yet.
